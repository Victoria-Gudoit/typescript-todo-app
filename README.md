## Installation

**Step 1:**
``` 
Make sure you have Node JS. Download or clone this repo 
```
**Step 2:**
``` 
Go to project root folder
``` 

**Step 3:**

Execute the following command in console to get the required dependencies: 

``` 
npm install
```
**Step 4:**

Run the app

``` 
npm run start
```
**Step 5:**

Build the app

``` 
npm run build
```
